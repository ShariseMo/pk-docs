# 引用组件

## 基础使用
`type` 引用了类型

`label` 文本

`number` 数字(数字引用)

`unit` 单位(数字引用)

`bgColor` 背景色

`brColor` 边框色

`color` 字体色

`icon` 图标(警告引用)

`showIcon` 是否带图标(警告引用)


<img src="/pk-quote/images/quote.jpg"/>

``` html
<template>
    <pk-quote
        type="warn"
        bgColor="#fff" 
        brColor="#fff" 
        color="#fff">
     </pk-quote>
</template>
```
