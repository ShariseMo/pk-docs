# 开关设置组件

## 基础使用
`icon` 图标

`bgColor` 背景色

`size` 图标大小

<img src="/pk-row-sp/images/row-sp.jpg"/>

``` html
<template>
    <pk-row-sp :icon="icon4" bgColor="#659be0" :size="70">
      <div slot="right-title"></div>
      <div slot="right-subtitle"></div>
      <div slot="left"></div>
    </pk-row-sp>
</template>
```
